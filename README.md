# ⚡ Что такое KnotTrace

**KnotTrace** — инструмент для разведки и визуального анализа инфраструктуры, который помогает обнаруживать скрытые связи между объектами и строить целостную картину исследуемой среды.

Вместо работы с хаотичными логами, таблицами и заметками вы получаете **интерактивный граф зависимостей**, где каждый объект становится частью общей инфраструктурной карты.

# 🚀 Возможности

## 🔍 Automated Recon

Автоматический сбор первичных данных и обнаружение связанных объектов.

KnotTrace помогает быстро получить стартовую карту инфраструктуры без необходимости ручной обработки большого объема информации.

---

## 🕸 Graph Intelligence

Связи между объектами отображаются в виде интерактивного графа.

Это позволяет обнаруживать:

* скрытые зависимости;
* цепочки взаимодействий;
* инфраструктурные кластеры;
* точки пересечения ресурсов;
* потенциальные направления исследования.

---

## 🎯 Deep Investigation

Каждый узел является полноценной точкой анализа.

Переходите между объектами, раскрывайте связи и постепенно углубляйтесь в исследуемую инфраструктуру.

---

## ⚡ Noise Reduction

Автоматизация позволяет быстро отделить значимые объекты от информационного шума и сосредоточиться на действительно важных элементах среды.

---

## 🧩 Dynamic Infrastructure Mapping

Граф расширяется по мере поступления новых данных.

В результате исследование превращается не в набор отдельных действий, а в построение полноценной карты инфраструктуры.

# 📦 Установка

### Клонирование репозитория

```bash
git clone https://github.com/marpelskiy/KnotTrace-Osint.git
cd KnotTrace
```

---

# 🚀 Запуск

### 1. Запуск backend

```bash
python server.py
```

### 2. Запуск интерфейса

Откройте:

```text
index.html
```

в браузере.

После запуска интерфейс автоматически подключится к серверу и отобразит граф исследуемой инфраструктуры.
