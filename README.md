# ⚡ Описание

**KnotTrace** — инструмент для визуального анализа инфраструктуры.

Он автоматически собирает данные, строит связи между объектами и отображает их в виде интерактивного графа.

👉 Вместо списков — структура
👉 Вместо данных — связи
👉 Вместо ручного анализа — визуальная карта

---

# 🕸 Как это работает

1. Вводите цель (домен / IP / объект)
2. Система собирает данные
3. Строит граф связей
4. Вы анализируете визуальную карту
5. Экспортируете результат

# 🚀 Установка

```bash
git clone https://github.com/marpelskiy/KnotTrace-Osint.git
cd KnotTrace
```

# ▶ Запуск

### 1. Запуск сервера

```bash
python server.py
```

### 2. Запуск интерфейса

Откройте файл:

```text
index.html
```

в браузере.
